# Инструкция по сборке EXE файла

## Быстрая сборка

### Вариант 1: EXE без консоли (рекомендуется для пользователей)
Запустите файл: **`build_exe.bat`**

Это создаст чистый EXE файл без окна консоли.

### Вариант 2: EXE с консолью (для отладки)
Запустите файл: **`build_exe_console.bat`**

Это создаст EXE файл с окном консоли, где будут видны все логи и сообщения.

## Что происходит при сборке

1. Автоматически устанавливается PyInstaller (если его нет)
2. Проверяются и устанавливаются все зависимости из `requirements.txt`
3. Очищаются старые файлы сборки
4. Создается EXE файл со всеми встроенными зависимостями

## Результат сборки

После успешной сборки вы найдете готовый EXE файл в папке **`dist`**:
- `dist\VRChat_Fishing_Bot.exe` - версия без консоли
- `dist\VRChat_Fishing_Bot_Console.exe` - версия с консолью

## ВАЖНО!

**Обязательно скопируйте файл `fishing_bot_settings.json` в ту же папку, где будет находиться EXE файл!**

Без этого файла бот не сможет сохранять настройки.

## Распространение

Для распространения бота вам нужно предоставить пользователям:
1. **VRChat_Fishing_Bot.exe** - главный исполняемый файл
2. **fishing_bot_settings.json** - файл с настройками

Пользователям НЕ нужно устанавливать Python или какие-либо зависимости!

## Требования для сборки

- Python 3.7 или выше
- pip (обычно идет вместе с Python)
- Интернет соединение (для загрузки PyInstaller при первом запуске)

## Размер файла

Итоговый EXE файл будет весить примерно 30-50 МБ, так как включает:
- Python интерпретатор
- Все библиотеки (numpy, pyaudio, pycaw, pywin32, и т.д.)
- Ваш скрипт

## Устранение проблем

### Ошибка при сборке
Если возникают ошибки:
1. Убедитесь, что Python установлен и добавлен в PATH
2. Запустите `install_dependencies.bat` для установки всех зависимостей
3. Попробуйте еще раз запустить build_exe.bat

### Антивирус блокирует EXE
Некоторые антивирусы могут считать созданный EXE подозрительным. Это нормально для EXE файлов, созданных с PyInstaller. Добавьте файл в исключения антивируса.

### EXE не запускается
1. Проверьте, что файл `fishing_bot_settings.json` находится в той же папке
2. Попробуйте версию с консолью (`build_exe_console.bat`) чтобы увидеть ошибки
3. Убедитесь, что в системе есть все необходимые DLL (обычно они есть в Windows 10/11)

## Ручная сборка (для продвинутых)

Если хотите настроить параметры сборки, используйте команду:

```bash
pyinstaller --onefile --windowed --name "VRChat_Fishing_Bot" vrchat_fishing_bot.py
```

Параметры:
- `--onefile` - один EXE файл (вместо папки с файлами)
- `--windowed` - без окна консоли
- `--console` - с окном консоли (для отладки)
- `--name` - имя выходного файла
- `--icon=path\to\icon.ico` - добавить иконку (опционально)
